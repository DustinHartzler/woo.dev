== Description ==

Easily calculate South African Post Office international parcel shipping rates with ability to add handling fees and insurance.

== Installation ==

1. Upload the folder 'woocommerce-sapo-international-parcel-service' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce SAPO International Parcel Service' through the 'Plugins' menu in WordPress
